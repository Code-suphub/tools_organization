## 核心设计特点

### 1. 色彩系统

#### 主色调
- **背景色**：纯白 `#ffffff`（浅色模式）/ 极黑 `#0d0d0d`（深色模式）
- **文字色**：深灰 `#242424` / 浅灰 `#e6e6e6`
- **次要文字**：中灰 `#6b6b6b` / `#a3a3a3`
- **分割线**：浅灰 `#e6e6e6` / 深灰 `#333333`

#### 强调色
- **主按钮**：Medium 绿 `#1a8917`（Sign up 按钮的颜色）
- **悬停色**：深绿 `#0e6f0d`

#### ❌ 禁止使用
- 渐变色
- 鲜艳的红蓝紫等颜色
- 复杂的色彩组合
- 玻璃态效果

#### 留白策略
- **宽松的行距**：1.6 - 1.8
- **充足的边距**：24px - 48px
- **内容最大宽度**：680px - 768px（居中显示）

### 3. 组件风格

#### 卡片设计
```css
/* 不使用阴影，使用边框 */
border: 1px solid #e6e6e6;
border-radius: 6px; /* 小圆角 */
background: #ffffff;

/* 悬停效果 */
&:hover {
  border-color: #c7c7c7; /* 边框变深 */
}
```

#### 按钮设计
```css
/* 主按钮（绿色） */
background: #1a8917;
color: #ffffff;
border-radius: 24px; /* 圆形胶囊按钮 */
padding: 8px 16px;
font-weight: 500;
text-transform: none; /* 不要全大写 */
box-shadow: none; /* 无阴影 */
```

```css
/* 次要按钮（边框按钮） */
border: 1px solid #e6e6e6;
color: #1a1a1a;
background: transparent;
```

#### 输入框设计
```css
border: 1px solid #e6e6e6;
border-radius: 24px; /* 圆形 */
padding: 8px 16px;
background: #f7f7f7; /* 浅灰背景 */

&:focus {
  border-color: #1a1a1a;
  background: #ffffff;
}
```

### 4. 字体系统

#### 字体族
- **UI 字体**：Inter, Noto Sans SC, system-ui
- **代码字体**：Fira Code, Consolas, monospace

#### 字体大小
```css
/* 标题 */
h1: 32px, font-weight: 700
h2: 24px, font-weight: 600
h3: 20px, font-weight: 600

/* 正文 */
body: 16px, line-height: 1.6
small: 14px, color: #6b6b6b
```

---

### 5. 图标风格

- **线性图标** - 使用 Material Icons Outlined 版本
- **统一粗细** - 所有图标 2px 线宽
- **统一色彩** - 继承文字颜色，不使用彩色图标

---

### 6. 动画效果

#### 极简过渡
```css
/* 只使用简单的颜色/边框过渡 */
transition: border-color 0.2s ease, color 0.2s ease;

/* 禁止使用 */
❌ transform: translateY()  /* 不要浮动效果 */
❌ transform: scale()        /* 不要缩放效果 */
❌ box-shadow 变化           /* 不要阴影动画 */
```

#### 悬停效果
```css
/* 链接悬停 */
a:hover {
  color: #1a1a1a;
  text-decoration: underline;
}

/* 按钮悬停 */
button:hover {
  background: #0e6f0d; /* 颜色变深 */
}

/* 卡片悬停 */
card:hover {
  border-color: #c7c7c7; /* 边框变深 */
}
```

## 与原设计的主要变更

| 特性 | 原方案（渐变风格） | 新方案（Medium 风格） |
|------|-----------------|---------------------|
| **色彩** | HSL 渐变色 | 黑白灰 + 绿色点缀 |
| **背景** | 浅灰 `hsl(0,0%,98%)` | 纯白 `#ffffff` |
| **卡片** | 渐变背景 + 阴影 | 边框 + 无阴影 |
| **按钮** | 多色渐变 | 绿色/边框按钮 |
| **圆角** | 12px | 6px（卡片）/ 24px（按钮）|
| **动画** | 浮动 + 阴影变化 | 仅颜色过渡 |
| **整体** | 炫酷科技感 | 内容至上极简 |

## 实施要点

✅ **保留**：
- React + MUI + Tailwind 技术栈
- 响应式布局
- 深色模式切换
- CodeMirror 代码编辑器

🔄 **调整**：
- 所有渐变色改为纯色
- 阴影效果改为边框
- 卡片悬停效果简化
- 按钮样式改为圆形胶囊
- 整体配色改为黑白灰

❌ **移除**：
- 玻璃态效果
- 复杂动画
- 多彩渐变
- 大圆角（卡片）
