# 工具网站功能需求汇总

## 文档概述

本文档基于以下四个工具网站的功能分析，整理了一个全面的开发工具网站需要包含的功能模块：

1. [开源中国在线工具](https://www.oschina.net/tool/date-converter) - IT Tools
2. [GroupDocs SQL比较工具](https://products.groupdocs.app/zh/comparison/sql-compare)
3. [草料二维码解码器](https://cli.im/deqr/other)
4. [华工具网站](https://www.huatools.com/)

---

## 功能分类汇总

### 1️⃣ JSON 处理类

#### 基础功能
- **JSON 格式化与美化** - 将压缩的 JSON 格式化为可读形式
- **JSON 压缩/最小化** - 移除空格和换行，压缩 JSON 体积
- **JSON 排序** - 按键名对 JSON 对象进行排序
- **JSON 校验** - 检查 JSON 语法是否正确

#### 高级功能
- **JSON 对比/差异检测** - 比较两个 JSON 文件的差异并高亮显示
- **JSON 合并** - 合并多个 JSON 对象

#### 转换功能
- **JSON ↔ YAML** - JSON 与 YAML 格式互转
- **JSON ↔ TOML** - JSON 与 TOML 格式互转
- **JSON ↔ XML** - JSON 与 XML 格式互转
- **JSON → URL 参数** - 将 JSON 对象转换为 URL 查询字符串
- **JSON → Dict/Python** - 将 JSON 转换为 Python 字典格式

---

### 2️⃣ SQL 处理类

#### 格式化工具
- **SQL 格式化** - 美化 SQL 语句，提高可读性
- **SQL 压缩** - 移除注释和多余空格，压缩 SQL 文件

#### 分析工具
- **SQL 对比工具** - 比较两个 SQL 脚本的差异
  - 逐字符/逐单词比较
  - 高亮显示添加/删除的内容
  - 支持粘贴文本或上传文件

---

### 3️⃣ 日期时间类

#### 时间转换
- **时间戳转换** - Unix 时间戳与人类可读时间互转
  - 支持多种格式：Timestamp、ISO 8601、ISO 9075、RFC 3339、RFC 7231 等
  - 支持本地化时区显示

#### 时间计算
- **日期计算器** - 计算两个日期之间的天数差
- **时间加减** - 在指定日期上增加或减少天数/小时等
- **ETA 计算器** - 预估剩余时间计算

#### 其他时间工具
- **秒表/计时器 Stopwatch** - 简单的计时工具
- **Crontab 生成器** - 可视化生成 Cron 表达式

---

### 4️⃣ 加解密/编码类

#### 哈希算法
- **MD5 加密** - 生成 MD5 哈希值
- **SHA 系列** - SHA-1、SHA-256、SHA-512 等
- **Hash 文本** - 对文本进行哈希处理
- **Hmac 生成器** - 生成 HMAC 认证码

#### 对称加密
- **文本加密/解密** - 使用对称加密算法加解密文本

#### 密码相关
- **bcrypt 生成器** - 生成 bcrypt 密码哈希
- **密码强度分析器** - 分析密码强度并给出建议
- **BIP39 助记词生成器** - 生成加密货币助记词

#### 密钥对生成
- **RSA 密钥对生成器** - 生成公私钥对

#### 编码转换
- **Base64 编码/解码** - 支持字符串和文件
- **URL 编码/解码** - URL 参数编码解码
- **Unicode 编码** - Unicode 与中文互转
- **ASCII 与二进制互转** - ASCII 码转换
- **HTML 实体编码** - HTML 特殊字符转义

---

### 5️⃣ 文本处理类

#### 格式转换
- **大小写转换** - 大写/小写/首字母大写等
- **驼峰与下划线互转** - CamelCase ↔ snake_case
- **全角半角转换** - 中英文标点转换

#### 文本分析
- **文本统计** - 字符数、单词数、行数统计
- **文本对比/Diff** - 比较两段文本的差异
- **词频统计** - 统计单词出现频率

#### 文本处理
- **文本排序** - 按字母/数字/长度排序
- **文本去重** - 移除重复行
- **行处理** - 添加前缀/后缀、删除空行等
- **文本压缩** - 移除多余空格和换行
- **字符串分割** - 按逗号、分号或自定义字符分割
- **文本转义/反转义** - 转义特殊字符

#### 正则工具
- **正则表达式测试器** - 实时测试正则表达式匹配

#### 生成工具
- **随机字符串生成** - 生成指定长度的随机字符串
- **Lorem Ipsum 生成器** - 生成占位文本
- **ASCII Art 生成器** - 生成 ASCII 艺术字

---

### 6️⃣ 代码格式化类

#### 多语言格式化
- **XML 格式化** - XML 美化和压缩
- **HTML 格式化** - HTML 代码美化
- **CSS 格式化** - CSS 代码美化和压缩
- **JavaScript 格式化** - JS 代码美化
- **YAML 格式化** - YAML 文件格式化

#### 转换工具
- **Markdown → HTML** - Markdown 转 HTML
- **XML → JSON** - XML 转 JSON

---

### 7️⃣ 开发工具类

#### 标识符生成
- **UUID 生成器** - 生成各版本的 UUID
- **ULID 生成器** - 生成时间排序的唯一标识符
- **Token 生成器** - 生成随机 Token

#### DevOps 工具
- **Docker Run → Docker Compose** - 将 docker run 命令转换为 compose 文件
- **Chmod 计算器** - 可视化计算 Linux 文件权限
- **Git 速查表** - Git 命令参考

#### 其他
- **抄袭检测** - 检测文本相似度
- **Benchmark 构建器** - 性能测试工具

---

### 8️⃣ 网络工具类

#### IP 工具
- **IP 地址查询** - 查询 IP 归属地信息
- **IPv4 子网计算器** - 计算子网掩码、可用主机数等
- **IPv4 地址转换器** - 不同 IP 格式转换
- **IPv6 ULA 生成器** - 生成 IPv6 唯一本地地址

#### 其他网络工具
- **MAC 地址生成器** - 生成随机 MAC 地址
- **端口扫描器** - 检测端口开放状态 (本地实现可能有限制)

---

### 9️⃣ 二维码/条形码类

#### 生成功能
- **二维码生成器** - 根据文本/URL 生成二维码
- **条形码生成器** - 生成各种格式的条形码

#### 解码功能
- **二维码解码器** - 从图片提取二维码信息
  - 上传图片解码
  - 输入图片 URL 解码
  - 摄像头实时扫描
  - 批量解码
  - 解码结果导出 (Excel/CSV/TXT)

---

### 🔟 图片处理类

#### 转换工具
- **图片转 Base64** - 将图片编码为 Base64 字符串
- **Base64 转图片** - 解码 Base64 为图片

#### 其他
- **Favicon 提取** - 从网站提取图标
- **颜色选择器** - 颜色代码转换 (HEX/RGB/HSL)

---

### 1️⃣1️⃣ 数学计算类

#### 基础计算
- **数学表达式求值** - 计算复杂数学表达式
- **百分比计算器** - 百分比计算工具

#### 数字处理
- **数字序列生成** - 生成等差/等比数列
- **数字求和** - 计算数字列表总和
- **数字格式化** - 添加千分位逗号、去除前导零等

---

### 1️⃣2️⃣ 文件处理类

#### PDF 工具
- **PDF 签名检查器** - 验证 PDF 数字签名

#### 文档对比
- **文档版本对比** - 跟踪代码和文档变更
- **翻译准确性验证** - 对比原文和译文

---

### 1️⃣3️⃣ 其他实用工具

#### 辅助工具
- **拼音转换** - 汉字转拼音
- **繁简转换** - 繁体字与简体字互转
- **Emoji 选择器** - 快速选择 Emoji 表情
- **NATO 字母表** - 北约音标字母转换
- **Numeronym 生成器** - 生成缩写词 (如 i18n)

#### 数据分析
- **CSV 数据处理** - CSV 文件解析和转换
- **列表计算** - List 转 Array/Tuple
- **字符串混淆器** - 混淆字符串内容

## 技术实现建议

### 前端技术栈
- **框架**: React / Vue.js / Next.js
- **UI 库**: Ant Design / Material-UI
- **样式**: Tailwind CSS / Styled Components
- **代码编辑器**: Monaco Editor / CodeMirror

### 功能实现优先级

#### P0 - 核心必备功能 (第一阶段)
1. JSON 格式化、校验、对比
2. 时间戳转换
3. Base64 编码/解码
4. URL 编码/解码
5. MD5/SHA 哈希
6. UUID 生成器
7. 文本对比
8. 二维码生成与解码

#### P1 - 重要功能 (第二阶段)
1. SQL 格式化与对比
2. 正则表达式测试
3. 颜色选择器
4. Cron 表达式生成
5. 各类格式化工具 (XML/HTML/CSS/JS)
6. 加解密工具套件

#### P2 - 增强功能 (第三阶段)
1. 图片处理工具
2. 文档对比工具
3. DevOps 辅助工具
4. 网络工具集
