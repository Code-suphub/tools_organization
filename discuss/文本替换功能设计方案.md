# 文本工具箱 - 文本替换功能设计方案 (已更新)

## 1. 背景
用户需要一个文本替换功能，能够将指定的字符串或模式替换为另一个字符串。例如将双引号 `"` 替换为转义的双引号 `\"`。

## 2. 功能目标
- [x] 支持自定义“查找”和“替换为”内容。
- [x] 支持一键执行替换。
- [x] 提供常用预设替换（转义引号、空格转逗号、换行转空格等）。
- [x] **统计信息移至顶部展示**。
- [x] **暂不提供正则表达式支持**。

## 3. UI 布局调整方案

### 3.1 顶部统计 (Header Stats)
- 将原本位于右侧的“文本统计”移至 `ToolCard` 的头部标题区域或紧随标题下方。
- 采用横向布局，展示核心指标：字数、行数、字数（Byte）等。

### 3.2 文本替换卡片 (Replacement Card)
- 在右侧功能面板新增一个独立卡片。
- **自定义替换区**:
  - `查找`: 单行输入框。
  - `替换为`: 单行输入框。
  - `全部替换` 按钮。
- **快速预设按钮组 (Quick Presets)**:
  - `引号转义 ( " -> \" )`
  - `引号恢复 ( \" -> " )`
  - `空格 -> 逗号`
  - `换行 -> 逗号`
  - `逗号 -> 换行`
  - `删除所有空格`

## 4. 逻辑处理
- 状态管理：使用 `findText` 和 `replaceText` 的 `useState`。
- 核心替换逻辑：
  ```javascript
  const handleReplace = (find, replace) => {
    if (find === undefined) find = findText;
    if (replace === undefined) replace = replaceText;
    if (!find && find !== ' ') return; // 允许空格作为查找内容
    const result = input.split(find).join(replace);
    setInput(result);
  };
  ```
