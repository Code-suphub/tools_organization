## 核心配色方案

### 浅色模式
```
背景：#f5f5f7（浅灰）
卡片：#ffffff（纯白）
主文字：#18181b（几乎黑）
次文字：#71717a（灰色）
强调色：#0066FF（蓝色 CTA）
边框：#e4e4e7（浅灰边框）
```

### 深色模式（已调亮）
```
背景：#27272a（Zinc-800，比之前亮）
卡片：#3f3f46（Zinc-700，比之前亮）
主文字：#fafafa（几乎白）
次文字：#a1a1aa（灰色）
强调色：#0066FF（蓝色 CTA）
边框：#52525b（深灰边框）
```

## 设计原则

✅ **极简专业** - 黑白灰为主，蓝色仅用于 CTA  
✅ **大量留白** - 舒适间距，视觉层次清晰  
✅ **克制配色** - 避免花哨，内容至上  
✅ **微妙动效** - 2px 浮起，极浅阴影  
✅ **现代圆角** - 20px 卡片圆角，8px 按钮圆角
